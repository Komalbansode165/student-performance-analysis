 Student Performance Analysis Project

This project analyzes marks of 25 students using Python and Pandas.

 Features
- Average marks of each subject  
- Maximum & minimum marks  
- Grade ordering  
- Added index column  
- Found class topper  
- Bar chart of average subject marks  
- Pie chart of grade distribution  

Tools Used
- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook  

 Files
- `students25.ipynb` – complete analysis  
- `students25.csv` – dataset
